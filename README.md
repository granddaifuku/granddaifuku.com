## Technologies

- [VitePress](https://vitepress.dev/)

## Developement

1. `make dev-site`
   This will create vitepress-dev docker image, and enter you to the bash of the image
2. `pnpm docs:dev --host` to display the site at localhost:5173

## Requirements

- Docker engine
